## 阶段详解

### 🔰 第一阶段：建立正确认知（3 课）

```
01_AI沟通的本质 ──▶ 02_大语言模型的能力边界 ──▶ 03_好指令的评估标准
    │                       │                           │
    ▼                       ▼                           ▼
 四大核心能力           五大常见错误                五大评估标准
 查·写·想·变          幻觉·混淆·自信              明确·具体·完整
 AI 能做什么           ·遗忘·格式错                ·可验证·可重复
```

**阶段目标**：建立对 AI 的正确认知，告别"AI 万能"和"AI 没用"两个极端。

**核心概念**：AI 的能力与限制、指令质量的评判标准

---

### 🛠️ 第二阶段：学会说人话（3 课）

```
04_明确目标与交付物 ──▶ 05_描述背景与受众 ──▶ 06_指令结构万能公式
    │                       │                       │
    ▼                       ▼                       ▼
  SMART 目标             背景五维度              RTBSF 框架
 三层目标结构           受众分析                Role → Task →
 交付物清单             表述调整                Background →
                                                Steps → Format
```

**阶段目标**：掌握写出高质量指令的核心技能。AI 第一次就能给出满意输出。

**核心概念**：SMART 原则、背景五维度、RTBSF 万能公式

---

### 🚀 第三阶段：高级技巧（6 课）

```
07_示例驱动 ──▶ 08_迭代式沟通 ──▶ 09_可复用模板
    │                │                  │
    ▼                ▼                  ▼
 正例·反例·对比   反馈·改进·升级    模板设计·管理·维护

10_系统化思考 ──▶ 11_防坑指南 ──▶ 12_高级技巧
    │                │                  │
    ▼                ▼                  ▼
 思维导图·分解    常见错误·检查清单   思维链·角色扮演

```

**阶段目标**：从"能写指令"升级到"能写出专家级指令"。

**核心概念**：示例驱动、迭代方法论、模板系统、思维链

---

### 🏆 第四阶段：实战应用（3 课）

```
13_实战项目 ──▶ 14_团队协作 ──▶ 15_评估与优化
    │               │               │
    ▼               ▼               ▼
 从零到一        统一规范         指令库体系
```

**阶段目标**：把全部知识转化为实际成果，建项目、带团队、建体系。

**核心概念**：AI Agent 设计、团队 AI 规范、个人指令库

## 学习建议

### 黄金原则

1. **不要跳课** — 每课都建立在上一课基础上，跳课 = 地基不稳
2. **学完就练** — 每课末尾都有练习，做完再继续
3. **做好笔记** — 记录你的最佳实践和踩坑经历
4. **教给别人** — 最好的学习方式是教会别人
